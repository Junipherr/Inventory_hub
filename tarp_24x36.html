System Overview

The Custodian Inventory Hub is a web-based system made to help custodians manage school items in one place. It keeps records organized, tracks item quantities, and makes it easier to monitor school property.

Core Objectives

  • Turn paper records into digital records
  • Keep item tracking clear and accurate
  • Make users responsible for assigned items
  • Organize inventory in a simple and structured way

System Modules

Inventory Management Stock Monitoring Secure Access Dashboard Overview

These modules allow users to add and update items, group them by category, and check stock levels quickly and easily.

Inventory Management

Users can add new items, update item details, and organize them by category. All records are stored properly so they can be viewed anytime.

Stock Monitoring

The system records items that come in and go out. It shows current stock levels and helps prevent shortages.
